CS 3650 - LLM Project

Just a study helper application. Allows management and reasoning of tasks.

Each task is defined as:
1. Name
2. Class
3. Body
4. Due Date

The application implements the following functionality:

1. Create Task
2. Update Task
3. Delete Task
4. Reason Task
5. Generate Study Plan

All tasks are loaded from tasks.json on load and saved into tasks.json on exit.
